# Vue Starter

Vue Starter is a Vue template based from @vue/cli.

## Vue Version

3.0.0

## Prerequisites

- Node.js (>= 12 required)
- npm package manager (>= 6 required)

## Commands

Compiles and hot-reloads for development.

```bash
npm run serve
```

Compiles and minifies for production.

```bash
npm run build
```

Run your unit tests.

```bash
npm run test:unit
```

Lints and fixes files.

```bash
npm run lint
```
